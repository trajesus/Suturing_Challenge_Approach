## Getting Started

### Prerequisites

This project requires:

- **Programming Language:** Python
- **Package Manager:** Pip

### Installation

1. **Clone the repository:**
```
git clone https://github.com/trajesus/Suturing_Challenge_Approach
```

2. **Navigate to the project directory:**
```
cd Suturing_Challenge_Approach/<year>
```
3. **Install dependencies:**
```
pip install -r OSS_25/Task1/requirements.txt OSS_25/Task2/requirements.txt
```

### Usage

Run the project with:
```
python {entrypoint}
```

### Testing

This project uses the `{test_framework}` test framework.

Run the tests with:
```
pytest
```
